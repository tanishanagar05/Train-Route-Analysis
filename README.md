# Train-Route-Analysis

## About
This project analyzes Indian train route data and predicts 
journey duration using Machine Learning (Linear Regression).

Internship Project at **Sysslan IT Solutions**

## Levels Completed
- ✅ Level 1 - Data Overview
- ✅ Level 2 - Data Cleaning & Feature Engineering  
- ✅ Level 3 - Data Exploration
- ✅ Level 4 - Visualization & Pattern Analysis
- ✅ Level 5 - Prediction Model Development
- ✅ Level 6 - Final Prediction System

## Dataset
- 1,86,074 rows | 11,113 trains | 8,147 stations

## Key Findings
- Average journey duration: 4.6 hours
- Busiest station: CST Mumbai
- Distance & duration correlation: 0.64
- Model R² Score: 0.45

## Tools Used
Python, Pandas, Matplotlib, Seaborn, Scikit-learn
